# LLM-Based Chatbot (HCI Group Project)

## Overview
Lightweight, open-source LLM chatbot for Human-Computer Interaction course project.

## Tech Stack
- Python
- Hugging Face Transformers
- FLAN-T5 (small)
- Gradio

## Run Locally

1. Create virtual environment:
   python -m venv .venv
   .venv\Scripts\activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python app.py
